# Specialized Agentic LLM Demo for JavaScript

## Overview
This interactive web application demonstrates the concept of specialized agentic Language Learning Models (LLMs) focused on the JavaScript ecosystem. It compares a general-purpose coding LLM with a specialized JavaScript agent LLM, highlighting how domain-specific training affects code generation.

## Features

- **Dual LLM Interface**: Side-by-side comparison of General Coding LLM vs Specialized JS Agent LLM
- **Interactive Prompting**: Test different coding tasks and see how each model responds
- **Vocabulary Restriction Control**: Adjust the specialization level of the JS Agent LLM
- **Educational Explanations**: Learn about key concepts like agentic behavior, reduced English vocabulary, and JS-derived languages
- **Responsive Design**: Works on both desktop and mobile devices

## Getting Started

1. Clone this repository
2. Open `index.html` in a modern web browser
3. Enter coding prompts and observe the differences in model outputs

## How It Works

The application simulates how a specialized LLM for JavaScript development would differ from a general-purpose coding LLM by:

1. **Domain Restriction**: The specialized model focuses solely on the JavaScript ecosystem (JS, React, Next.js, Node.js)
2. **Vocabulary Constraints**: Uses a reduced English vocabulary optimized for technical documentation
3. **Code Style**: Prefers modern JavaScript syntax and common patterns from the JS ecosystem

## Development
